# School Management System

A web-based school management system built with PHP and MySQL.

## Features

- Admin panel for managing students, teachers, staff, and classes
- Student portal for viewing profiles and results
- Examination and marks management
- News and announcements system
- Public-facing pages for school information

## Installation

1. Update database credentials in `includes/dbconnection.php`
2. Deploy files to web server
3. Access admin at `/admin/login.php` and student portal at `/user/login.php`

## Requirements

- PHP 7.0+
- MySQL 5.6+
- Apache/Nginx

## Structure

- `/admin` - Admin panel
- `/user` - Student portal
- `/includes` - Database connection and common files
- `/css`, `/js` - Assets
- `/images`, `/img` - Media files
